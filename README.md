## 문제사항

- add.hide // remove.hide 적용이 안됌. (영화 검색 로딩 애니메이션 & 검색 더보기 버튼)
- 영화 검색 시 포스터, 제목, 출시년도 중 년도 데이터 못읽음 ->undefined
- 영화를 검색하고 'view more'버튼으로 다음 페이지를 불러는것 까지는 작동. 이후에 같은 영화목록이 무한대로 불러와짐
